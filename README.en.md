# virt_iio_device_driver

#### Description
实现一个虚拟的iio device driver，该iio device提供两路温度检测，支持读取当前的温度值，并且支持温度阈值告警中断，且支持连续数据采集功能，当温度数据可读时，可产生可读中断。
